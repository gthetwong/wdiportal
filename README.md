# WDI Portal

Developed during SF WDI Jan 2014 Cohort by @kaplanoah. 

This app is a centralized place for WDI students, instructors, and
procuders. It allows all parties to post and read:

* Gists
* Resources
* Student projects
* Events

## Functionality coming soon
